# 核心指标拆解分析

根据两张业务截图整理的完整静态汇报页面，展示收入、营业额、到手率、客单价、有效订单及三个平台的下单与转化链路。

## 使用方式

- 直接双击 `index.html` 可离线查看。
- 本地预览可在当前目录运行 `python3 -m http.server 8768`，再打开终端打印的本地地址。
- 数据、样式和渲染脚本均内置在 `index.html`，不依赖网络字体、外部图片或第三方脚本。

## 文件说明

- `index.html`：完整静态页面。
- `verify-page.mjs`：数据与结构校验器。
- `validation.json`：多尺寸预览验证结果。
- `preview.png`：桌面端预览图。
- `.nojekyll`：GitHub Pages 静态发布标记。

## 数据说明

所有数值按原始截图录入，未对缺失数据进行推测。三个平台依次为美团外卖、淘宝闪购和京东外卖。

## 隐私提示

页面包含业务经营数据。源代码仓库默认保持私有；启用 GitHub Pages 后应根据账号能力确认站点实际访问范围。
